# /devlog - 일일 개발 로그 작성

오늘 날짜로 DEVLOG.md에 개발 로그를 작성해줘.

## 작업 절차

### 1. 오늘 커밋 확인
```bash
git log --oneline --since="midnight" --until="now"
```

### 2. DEVLOG.md 업데이트
파일 상단(제목 바로 아래)에 오늘 날짜의 엔트리 추가:

```markdown
## {오늘 날짜 YYYY-MM-DD}
### 작업 내용
- {커밋 메시지 기반 작업 내용}
- {추가 작업 내용}

### 학습/메모
- {오늘 배운 것, 기억할 것}

### 이슈/해결
- {발생한 문제와 해결 방법}

---
```

## 주의사항
- 같은 날짜 엔트리가 이미 있으면 내용만 추가/업데이트
- 커밋이 없어도 수동으로 작업 내용 입력 가능
- 학습/메모, 이슈/해결 섹션은 내용 없으면 `-` 유지
